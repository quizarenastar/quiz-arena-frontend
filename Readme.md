# Quiz Arena Frontend

A React-based frontend for the Quiz Arena application.

## Features

-   User authentication
-   Quiz creation and participation
-   Real-time leaderboard
-   Responsive design

## Getting Started

### Prerequisites

-   Node.js (v18+)
-   npm or yarn

### Installation

```bash
git clone https://github.com/yourusername/quiz-arena-frontend.git
cd quiz-arena-frontend
npm install
```

### Running the App

```bash
npm start
```
